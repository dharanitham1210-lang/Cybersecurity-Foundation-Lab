# Wireshark

## Introduction

Wireshark is a free and open-source network protocol analyzer. It captures and analyzes network packets in real time, making it one of the most widely used tools for network troubleshooting and cybersecurity analysis.

---

# Features

- Captures live network traffic
- Analyzes network protocols
- Supports hundreds of protocols
- Packet filtering
- Export captured packets

---

# Installation

Wireshark can be installed on Windows, Linux, and macOS.

Kali Linux:

```bash
sudo apt install wireshark
```

---

# Basic Steps

1. Open Wireshark.
2. Select a network interface.
3. Click **Start Capturing Packets**.
4. Browse a website or generate network traffic.
5. Stop the capture.
6. Apply filters such as:

```text
http
tcp
dns
icmp
```
