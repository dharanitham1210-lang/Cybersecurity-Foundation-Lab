# Networking Fundamentals

## Introduction

Networking is the process of connecting computers and devices to communicate and share data. Understanding networking is essential in cybersecurity because most attacks and defenses occur over networks.

---

# OSI Model

The OSI (Open Systems Interconnection) Model consists of seven layers.

| Layer | Function |

| 7. Application | Provides services to end users |
| 6. Presentation | Data encryption and formatting |
| 5. Session | Manages communication sessions |
| 4. Transport | Reliable data transfer (TCP/UDP) |
| 3. Network | Routing using IP addresses |
| 2. Data Link | MAC addressing and switching |
| 1. Physical | Transmission of bits through cables |

---

# TCP/IP Model

The TCP/IP Model contains four layers.

- Application Layer
- Transport Layer
- Internet Layer
- Network Access Layer

---

# Common Network Protocols

### HTTP
- Port: 80
- Used for web communication.

### HTTPS
- Port: 443
- Secure version of HTTP using SSL/TLS.

### DNS
- Port: 53
- Converts domain names into IP addresses.

### FTP
- Port: 21
- Used for file transfer.

### SSH
- Port: 22
- Secure remote login.

---

# IP Address

An IP address uniquely identifies a device on a network.

Example:
192.168.1.10

Types:
- Public IP
- Private IP

---

# Subnetting

Subnetting divides a network into smaller networks to improve performance and security.

---

# NAT (Network Address Translation)

NAT allows multiple private devices to share a single public IP address.

---

# Conclusion

Networking forms the backbone of cybersecurity. A strong understanding of networking concepts helps identify vulnerabilities, analyze attacks, and secure communication.
