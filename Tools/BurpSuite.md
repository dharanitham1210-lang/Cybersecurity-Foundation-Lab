# Burp Suite

## Overview

Burp Suite is a web application security testing tool developed by PortSwigger. It is widely used by penetration testers and security professionals to intercept, inspect, modify, and analyze HTTP/HTTPS requests and responses between a browser and a web server.

---

## Features

- Intercept HTTP/HTTPS requests
- Proxy support
- Repeater
- Intruder
- Decoder
- Comparer

---

## Installation

### Kali Linux

```bash
sudo apt install burpsuite
```

Or download the Community Edition from the official PortSwigger website.

---

## Basic Workflow

1. Launch Burp Suite.
2. Configure the browser proxy.
3. Turn on Intercept.
4. Browse a website.
5. Capture and inspect requests.
6. Modify requests and forward them.

---
