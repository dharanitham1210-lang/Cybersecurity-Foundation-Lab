# Nmap

## Overview

Nmap (Network Mapper) is an open-source network scanning tool used to discover hosts, identify open ports, detect running services, and perform security assessments.

---

## Features

- Host Discovery
- Port Scanning
- Service Version Detection
- OS Detection
- Network Inventory

---

## Installation

```bash
sudo apt install nmap
```

---

## Common Commands

### Basic Scan

```bash
nmap 192.168.1.10
```

### Service Version Detection

```bash
nmap -sV 192.168.1.10
```

### OS Detection

```bash
nmap -O 192.168.1.10
```

---

## Applications

- Network Auditing
- Vulnerability Assessment
- Device Discovery
- Security Testing
