# Wireshark

## Overview

Wireshark is a free and open-source network protocol analyzer used to capture, inspect, and analyze network traffic in real time. It is one of the most widely used tools in cybersecurity and network troubleshooting.

---

## Features

- Live packet capture
- Protocol analysis
- Packet filtering
- Deep packet inspection
- Export captured packets

---

## Installation (Kali Linux)

```bash
sudo apt update
sudo apt install wireshark
```

---

## Basic Usage

1. Open Wireshark.
2. Select the network interface.
3. Click **Start Capture**.
4. Generate network traffic.
5. Stop the capture.
6. Analyze packets using filters.

Common Filters:

```text
http
tcp
udp
dns
icmp
```
